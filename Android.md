#### 统计
| 组件名                   | 作用        | 是否有界面 | 常见用途          |
| --------------------- | --------- | ----- | ------------- |
| **Activity**          | 展示界面、用户交互 | 有     | 登录页、设置页、聊天页面等 |
| **Service**           | 执行长时间后台任务 | 无     | 下载、播放音乐、同步数据  |
| **BroadcastReceiver** | 响应广播事件    | 无     | 监听网络、电池、短信等   |
| **ContentProvider**   | 跨应用共享数据   | 无     | 联系人、图片、音视频访问  |
